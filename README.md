# Codesis FrontEnd Assignment

A **modern, responsive, and interactive website** built with **React and Tailwind CSS**.  
This project contains **2 main pages** – Home, UsersPage – each crafted with **smooth animations, clean layouts, and engaging UI design**.  

---

## 👤 User Dashboard

### Home Page

The **Home Page** consists of list of users.  
The implemented features are :-
- **Debounced Search Input** .  
- **Sorting Names Alphabetically** .
- ⚡ **Mobile Responsive**.    
- **No Page Found** - if user routes to a different url, it has Page Not Found.  
- **Deployed on netlify** - Temporarily deployed.   

## ⚡ How to Run Locally
```bash
# Clone the repository
git clone https://github.com/Tanmay-0412/Codesis_Assignment.git

# Move into the project directory
cd codesis_assignment

# Install dependencies
npm install

# Start the development server
npm run dev
```
